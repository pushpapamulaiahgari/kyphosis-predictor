# Kyphosis Prediction App

A Streamlit web app that predicts the risk of Kyphosis after spinal surgery using a Logistic Regression model.

 **Features**
- User inputs: Age in months, Number of vertebrae involved, Start level
- Output: `PRESENT` or `ABSENT` with Risk % 
- Built with: Python, Scikit-learn, Streamlit

### **How to Run Locally**
```bash
pip install -r requirements.txt
streamlit run app.py
